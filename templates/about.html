About Westfall Personal Assistant

Overview

Westfall Personal Assistant is an AI-powered desktop application that provides intelligent assistance for various tasks. Built with modern web technologies and local AI models, it offers privacy-focused AI assistance without requiring cloud connectivity.

Getting Started

To get started with Westfall Personal Assistant:

  1. Download a compatible GGUF model file
  2. Place it in the models/ directory
  3. Configure the model path in Settings
  4. Start using your personal AI assistant!
